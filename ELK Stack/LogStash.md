# Definition
* Logstash = Logstash is a server‑side data processing pipeline that ingests data from multiple sources simultaneously, transforms it, and then sends it to a "stash" like Elasticsearch

# To Read
* Logstash collects logs from different sources ( SysLogs, Log files , Kafka Queues … etc ) and stores logs in Elasticsearch then Kibana makes a query to elastic search to view logs in UI 
* It is data pipeline tool. 
* Open source tool for log analysis 
* It is a data pipeline to elastic search 
* It collects and monitor logs from remote machine 
* Collects Logs , Sends Logs to Elastic search through data pipeline 
* Logstash 5.6.5 stable version
     * It will centralize the data processing. 
     * It will collect , parse & analyze the structured / unstructured data & events. 
     * It provides plugin to connect to various types of input sources & platforms.
